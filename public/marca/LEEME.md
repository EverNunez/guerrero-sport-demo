# Marca / logos oficiales — Guerrero Sport

Estos son los logos oficiales del cliente. Generados a partir de los PDF originales.

| Archivo | Qué es | Dónde se usa |
|---|---|---|
| `casco.png` | Emblema: casco espartano con "SPORT" | Navbar, hero, favicon, OG |
| `logo-guerrero.png` | Wordmark "Guerrero Sport" (texto negro) | Fondos claros |
| `logo-guerrero-white.png` | Wordmark en blanco | Fondos oscuros (el sitio es oscuro) |
| `casco.pdf` | Vector original del emblema | Respaldo / impresión |
| `logo-guerrero.pdf` | Vector original del wordmark | Respaldo / impresión |

## Si el cliente envía un logo nuevo

1. Reemplazá el PNG correspondiente manteniendo el mismo nombre, **o**
2. Subí el archivo nuevo y actualizá la ruta en `components/ui/Logo.jsx`
   (emblema + wordmark) y, si cambia el favicon, regenerá los íconos de `public/`.

> Los PNG tienen fondo transparente. El wordmark blanco es el que se ve sobre el
> fondo oscuro del sitio.
